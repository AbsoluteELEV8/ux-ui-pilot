## [0.1.0] — 2026-02-23

### Added

- Initial release: UX/UI design agent
- 6 capabilities: Design System Generator, Component Architecture, Accessibility Audit, User Flow Mapper, Wireframe Advisor, Design Critique
- MCP server with 6 registered tools for IDE integration
- CLI interface with 6 subcommands
- Knowledge base: 50 WCAG 2.2 criteria, 10 Nielsen heuristics, 18 UI patterns, design token templates
- Claude-powered design reasoning via LLMClient
- Structured output in Markdown and JSON formats
- Full 4 .md pattern (CLAUDE.md, cursor rules, skills, subagent)
- Workflow orchestration rules
- Reference documents: WCAG checklist, Nielsen heuristics guide, UI patterns catalog
- Task tracking infrastructure
